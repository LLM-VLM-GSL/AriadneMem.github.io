Abstract

Long-horizon LLM agents require memory systems that remain accurate under fixed context budgets. However, existing systems struggle with disconnected evidence and state updates. We propose AriadneMem, a structured memory system that addresses these failure modes via a decoupled two-phase pipeline. On LoCoMo experiments, AriadneMem improves Multi-Hop F1 by 15.2% and Average F1 by 9.0% over strong baselines, using only 497 context tokens.
