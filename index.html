Abstract

Long-horizon LLM agents require memory systems that remain accurate under fixed context budgets. However, existing systems struggle with disconnected evidence and state updates. We propose AriadneMem, a structured memory system that addresses these failure modes via a decoupled two-phase pipeline. In the offline phase, we use entropy-aware gating and conflict-aware coarsening. In the online reasoning phase, we execute algorithmic bridge discovery to reconstruct missing logical paths. On LoCoMo experiments, AriadneMem improves Multi-Hop F1 by 15.2% while reducing runtime by 77.8% using only 497 context tokens.
