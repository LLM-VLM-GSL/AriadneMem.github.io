Abstract

Long-horizon LLM agents require memory systems that remain accurate under fixed context budgets. However, existing systems struggle with disconnected evidence and state updates. We propose AriadneMem, a structured memory system that addresses these failure modes via a decoupled two-phase pipeline. By offloading reasoning to the graph layer, AriadneMem reduces total runtime by 77.8% using only 497 context tokens.

AriadneMem Architecture

The pipeline is decoupled into two phases: (I) Offline Memory Construction and (II) Online Structural Reasoning.
